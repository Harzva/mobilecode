# MobileCode V12 — 记忆/习惯/性能/二进制分析

## 1. 记忆管理 + 个人习惯管理

### 记忆管理 (Memory Management)
AI记住的东西需要可视化：
- 项目知识库（索引的文件、函数、类）
- 代码偏好（命名规范、缩进风格、注释习惯）
- 对话历史（与AI的聊天记录）
- 错误模式（经常犯的错、常见修复）
- 常用代码片段（高频使用的代码）

### 个人习惯管理 (Habit Management)
用户的编码习惯追踪：
- 编码时间分布（几点最活跃）
- 常用语言排行
- 代码风格偏好
- 快捷键使用
- 项目类型偏好
- 每周/月编码统计

## 2. 性能模式 (Performance Mode)

两种模式：

| 维度 | 流畅模式 (默认) | 性能模式 |
|------|----------------|----------|
| 目标 | 极致流畅 | 全功能开放 |
| 粒子效果 | ❌ 关闭 | ✅ 开启 |
| 实时波形 | ❌ 关闭 | ✅ 开启 |
| 语法高亮 | ✅ 基础 | ✅ 完整 |
| AI多Agent | ❌ 单Agent | ✅ 多Agent |
| 实时预览 | ❌ 手动触发 | ✅ 自动刷新 |
| 动画效果 | ❌ 减少 | ✅ 全部 |
| 后台深潜 | ✅ 基础 | ✅ 完整 |
| 内存占用 | 低 (<150MB) | 高 (<400MB) |

用户可以自定义每个功能的开关。

## 3. 二进制分析版本 (Binary Analysis)

只分析不执行：
- APK/IPA 反编译分析
- 二进制文件结构解析
- 依赖分析
- 安全扫描
- 代码质量审计
- 静态分析（无需运行时）
